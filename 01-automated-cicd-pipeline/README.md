## 📝 Project Overview
This project automates the CI/CD (Continuous Integration/Continuous Deployment) pipeline using AWS services and Docker to ensure smooth, automated deployment and integration of applications.

---

## 🛠️ Tech Stack
* **Cloud Platform:** AWS (EC2, ECR, ECS)
* **Containerization:** Docker
* **Pipeline Automation:** GitHub Actions / AWS CodePipeline
* **Version Control:** GitHub

---

## 🚀 Features
* **Automated Testing:** Code is automatically tested on every push.
* **Docker Support:** Applications are packaged into lightweight Docker containers.
* **Continuous Deployment:** Smooth and seamless zero-downtime deployment to AWS.
